# Codeshare_Hect2026
Analysis code for “Acoustic regularities define perceptual and cortical representations of voice-likeness.” Includes acoustic, behavioral, and intracranial EEG analysis pipelines.

This repository provides the complete analysis code used in the study “Acoustic regularities define perceptual and cortical representations of voice-likeness.” The goal of the project is to identify which acoustic dimensions support graded perceptual judgments of voice-likeness and how these dimensions are represented in human auditory cortex using intracranial recordings.
